## ✨ What is PRIV web?

**PRIV web** is a growing collection of **utility tools** that work entirely inside your browser —  
zero data leaves your device, zero trackers, zero server logs.  

Built because I got tired of big tech vacuuming every click, every file, every character.  
Every tool is **100% client‑side**, **always free**, and **never asks for your data**.

## 🔧 Why PRIV web is different

|                               | Big Tech Tools | PRIV web |
|-------------------------------|----------------|----------|
| ❌ Sends data to server       | ✅ Yes         | ❌ No    |
| ❌ Tracks your behavior       | ✅ Yes         | ❌ No    |
| ❌ Requires sign‑up / login   | ✅ Often       | ❌ No    |
| ❌ Shows ads / affiliate links | ✅ Often       | ❌ No    |
| ✅ Runs offline after load    | ❌ Rarely      | ✅ Yes   |

Every tool:
- 🧠 Processes everything **locally** (JavaScript in your browser)
- 📁 Never uploads files or text
- 📊 No analytics, no telemetry, no cookies
- 💸 **Forever free** – no paywalls, no "premium"
